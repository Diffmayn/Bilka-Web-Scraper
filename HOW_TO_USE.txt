STEP 1: Scrape Products
────────────────────────
Double-click this file:
    📄 run_manual_scrape.bat

This will:
  • Prompt you to select category (Electronics/Home/Fashion/Sports)
  • Ask how many products to scrape (10-200)
  • Automatically scrape from Bilka.dk
  • Store all data in database

Result: ✅ 20 products scraped and stored


STEP 2: View Results in Dashboard
──────────────────────────────────
Double-click this file:
    📄 run_local_dashboard.bat

This will:
  • Open Streamlit dashboard at http://localhost:8501
  • Show all scraped products
  • Display anomaly scores
  • Let you export to CSV

Result: ✅ Interactive dashboard with all data
