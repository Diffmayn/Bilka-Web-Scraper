═══════════════════════════════════════════════════════════════════════
                🛠️  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

❌ "No products found"
   → Make sure Chrome is installed
   → Check config/settings.yaml: headless: false
   → Run: python test_chrome.py

❌ "ChromeDriver error"
   → Download matching ChromeDriver version
   → Place in: drivers/chromedriver.exe

❌ "Port 8501 in use"
   → Close other Streamlit instances
   → Or use: streamlit run streamlit_app.py --server.port 8502
